# OpenClaw Social Media Skill

## 简介
社交媒体营销自动化 Skill，帮助出海独立开发者自动化运营 X (Twitter)、Facebook、微信公众号。

## 功能
- **内容发布**: 定时自动发布内容到各平台
- **评论管理**: 监控评论并 AI 自动生成回复
- **主动互动**: 搜索相关内容进行互动获客
- **数据分析**: 生成发布效果和转化报告

## 命令

### 发布内容
```
@openclaw 发布一条关于 [主题] 的推文
@openclaw 生成今日发布内容
@openclaw 发布到所有平台
```

### 评论管理
```
@openclaw 检查新评论
@openclaw 回复评论 [评论 ID]
@openclaw 评论日报
```

### 主动互动
```
@openclaw 搜索 [关键词] 并互动
@openclaw 今日互动任务
@openclaw 互动周报
```

### 数据分析
```
@openclaw 发布数据
@openclaw 周报
@openclaw 月报
```
