# CODEBUDDY 聊天 skills claude同源: 爆文起号全行业解决方案

## 一、核心技能架构

dymp 项目包含两个智能化技能: **dytxt**(抖音内容解析) 和 **singletxt2md**(文本转Markdown)。两者协作形成完整的内容生产流水线,实现从视频分享到爆款文案的全流程自动化。

---

## 二、dytxt 技能: 智能视频内容解析

### 触发机制
当用户粘贴包含抖音URL的分享文本时自动激活,支持两种URL格式:
- `https://v.douyin.com/xxxxx/` (短链接)
- `https://www.douyin.com/video/xxxxx` (完整链接)

### 核心能力

#### 1. 智能信息提取
- 自动解析视频ID: 从URL精准提取视频唯一标识
- 提取视频标题: 保留完整标题包括话题标签(如`#魔兽世界#时光服`)
- 提取作者信息: 识别`@xxx`格式作者名,解析失败使用默认值`@作者1`
- 提取发布时间: 如"9.25"等日期信息

#### 2. 单篇/多篇模式智能切换
- **单篇模式**: 处理单个视频,输出`single_{视频ID}.txt`
- **多篇模式**:
  - 自动识别多个视频分享(用空行分隔)
  - 智能文件编号管理: 扫描`output/txt/`目录已有文件
  - 今日无文件: 创建`temp_{日期}_1.txt`
  - 今日有文件: 询问追加(最大num)或新建(num+1)

#### 3. 标准化输出格式
```txt
# id: 7583909568174935342
# url: https://www.douyin.com/video/7583909568174935342
# title: 三系崛起!时光服武器战加强来啦
# author: "@喵的wow世界"
# content: (留空,待用户填充)
```

### 使用示例

#### 场景1: 单篇视频分享
```
用户: 9.25 J@V.yt 05/26 FuS:/ 泄密局档案 | 时光服P2饰品特效... https://v.douyin.com/7gvf5ENi5fQ/

CODEBUDDY:
检测到抖音分享内容,已解析:
- 视频ID: 7gvf5ENi5fQ
- 标题: 时光服P2饰品特效,橙装升级,装备重制
- 作者: @喵影三三
- 发布时间: 9.25

请提供输出文件名(留空则自动生成 single_7gvf5ENi5fQ.txt):
```

#### 场景2: 多篇视频分享
```
用户: (提供多个抖音分享文本,用空行分隔)

CODEBUDDY:
检测到 {N} 个视频分享内容。

今天日期: 2026-01-25
已扫描今日文件: temp_2026-01-25_1.txt, temp_2026-01-25_2.txt
最大编号: 2

请选择输出方式:
1. 追加到现有文件: output/txt/temp_2026-01-25_2.txt
2. 创建新文件: output/txt/temp_2026-01-25_3.txt

请回复 1 或 2(默认为2):
```

## 三、singletxt2md 技能: 智能内容转Markdown

### 触发机制
用户提供包含`# id:`、`# title:`、`# content:`的txt文件路径或内容时激活。

### 核心能力

#### 1. 智能领域识别系统(5级优先级)
- **优先级1**: 用户明确指定领域(如"编程/java")
- **优先级2**: AI大模型智能识别(基于标题特征分析)
  - 示例: "时光服武器战" → 识别为魔兽世界
  - 示例: "Spring Boot自动装配" → 识别为编程/java
- **优先级3**: 向用户询问确认识别结果
- **优先级4**: 接受用户修正领域
- **优先级5**: 无法识别时询问用户

#### 2. 专业术语自动修正
- 根据领域加载对应术语字典:`keywords/{领域}/noun/*_key.txt`
- 支持领域: 魔兽世界、编程(java/python/react/vue等)
- 魔兽世界修正示例:
  - "星火数" → "星火术" (技能名)
  - "成群" → "虫群" (技能名)
  - "时光伏" → "时光服" (服务器名)

#### 3. 内容结构化处理
- 使用一级标题`# 主题1`分段
- 每段150-200字,自动合并短段落
- 修正错别字、专业术语
- 保持原文口语化风格
- 屏蔽作者信息避免版权问题
- 删除广告营销内容

#### 4. 爆款标题自动生成
- 在验证清单后生成10个不同类型公众号爆款标题
- 涵盖疑问式、数字式、痛点式等类型

#### 5. 验证检查清单
确保输出质量:
- [✓] 所有文字都定义了一级标题
- [✓] 段落与原文匹配度>90%
- [✓] 术语修正清单(如适用)
- [✓] 可复制的markdown格式

### 使用示例

## 四、完整爆款起号流程

### 阶段1: 内容收集(dytxt)
```
用户粘贴抖音分享文本
    ↓
dytxt自动解析URL/ID/标题/作者
    ↓
生成标准化txt文件到output/txt/
    ↓
用户观看视频,手动填充content内容
```

### 阶段2: 内容加工(singletxt2md)
```
用户提供txt文件路径
    ↓
singletxt2md智能识别领域
    ↓
加载对应术语字典
    ↓
执行术语修正+结构化处理
    ↓
生成Markdown到mp/目录
    ↓
自动添加验证清单+10个爆款标题
```

### 阶段3: 内容发布
- 直接复制mp/目录下.md文件
- 选择合适爆款标题
- 发布到公众号/知乎/小红书等平台
