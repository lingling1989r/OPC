# 一、Claude Skills的价值和优势

## 大幅提升写作效率
建议所有人都去学Claude Code Skills，真的太香了。之前写篇文章，从折腾选题到搜集资料，再到写初稿、改稿、校对，就算是专业的创作者至少也要两小时起步。现在将大博主都在用的写作流程提炼成SOP，再让Claude帮我写成Skills，就可以让AI帮你自动化写作和归档。一篇高质量的文稿，从之前的两小时起步到20分钟就能搞定。

# 二、Claude Skills实战：自动化写作Agent

## 写作流程拆解
在开始创建我们的写作Skill之前，我们需要拆解下我们的工作流程，分为理解需求、信息搜集、选题确认、创建任务拆解、学习我的风格、初稿创作、匠AI这几个步骤。第一步，理解需求，按照格式自动保存项目brief文件夹。第二步，信息搜集与知识管理。因为涉及到科技产品测评，所以多渠道的信息搜集是必须要做的，搜索完了自动保存到对应文件夹，内容必须包含信息收集时间和信息来源。



**来源**: #id 7596229808447573282 | 点赞: 4265 | @Jacky无限生长

## 子观点3：风格学习和初稿创作
学习风格，参考文件夹中历史文章的风格，提取开头方式、结构偏好、语言特征、金句风格等内容。根据我的风格加入实践案例进行初稿创作。初稿无需特别完美，咱们让他先把框架先搭出来。

**来源**: #id 7596229808447573282 | 点赞: 4265 | @Jacky无限生长

## 子观点4：三重审核机制
让它进行三重审核。就像在媒体工作一样，初稿是小编写的，刊登前要经过主编的一层层审核，从数据校对到去AI化再到细节打磨，让初稿成为直接能用的高质量终稿文档。

## 子观点2：创建写作Agent的具体步骤
我同样整理在知识库了，直接把文档上传到Claude，让他理解内容，自己执行。你只等待十几分钟就创建好了。创建好的skill文档存放目录，在这个文件夹你可以打开确认文件是否符合skill的标准。Skill到MD是必须有的，重点关注name description这两个元数据里面包含了AI自动触发的要求。
